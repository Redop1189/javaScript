### Primitive Types -
3. Primitive Types

### Reference Types -
4. Objects
```javascript
let person = {
  name: 'John',
  age: 20
};
// For selection
// Dot Notation
person.name = 'Harry';

// Bracket Notation
person['name'] = 'Harry';
```
